# MCalc
### MCalc - calculator for performing simple mathematical operations in all existing number systems. 
Installation guide for linux:
1)  install cmake
2)  git clone https://github.com/nesterovmaxim31/MCalc
3)  cd MCalc
4)  cmake . && make && ./MCalc
